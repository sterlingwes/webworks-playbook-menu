Toggle PBmenu

This is a basic skeleton jqUi sample showcasing pbmenu functionality.

No links or buttons are supposed to work except for 'Toggle Nav' which will show / hide the PlayBook menu.

You'll need to compile this as a BlackBerry PlayBook Webworks app if you want to test the bezel swipe-down for the menu. The bezel-swipe will not work in a PlayBook browser!

By default, the menu will disappear when a menu button is clicked. It will also disappear if the user clicks/touches outside of the menu itself.

For more detail on available methods and configuration properties, click here.
